location_dev.js

¶

Все урлы должны заканчиваться "/", кроме тех что указываются в allowedPaths

Доступные параметры: app (не обязательный параметр) - адрес куда приложение будет посылать все запросы, если сервисы находятся не рутовом урле; Пример: locations: { app: 'http://localhost/bauhaus/' }

static (не обязательный параметр) - адрес откуда должны грузится все файлы, если он отличается от того что указан в app; Пример: locations: { app: 'http://localhost/bauhaus/', static: 'http://localhost/bauhaus/Content/' }

allowedPaths (не обязательный параметр) - массив состоящий из pathname на которых приложение может запускаться относительно корня указанного в app. Пример: locations: { app: 'http://localhost/bauhaus/', allowedPaths: ['/', '/bauhaus'] }
